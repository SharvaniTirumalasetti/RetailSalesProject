# Retail Sales Project

This project analyzes retail sales data using **SQL** for data cleaning and **Excel** for creating an interactive dashboard.  
The goal is to identify key business insights such as sales trends, top products, and regional performance.

## Features
- Cleaned and transformed raw data using SQL.
- Built an Excel dashboard with:
  - KPIs (Total Sales, Profit, Quantity, Avg. Order Value, etc.)
  - Monthly sales trends.
  - Top products by sales & profit.
  - Regional performance analysis.
- Insights provided in a clear and interactive format.
